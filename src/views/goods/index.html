  1. 重量 Carat
  2. 　　钻石的重量是用克拉来计量。
    　　一克拉等于0.2g，每一克拉分为一百分（或相当于200毫克），因此一颗35分的钻石重0.35克拉。
    　　同等品质的钻石，重量越大越珍贵。

  1. 净度 Clarity
  2. 　　净度代表钻石的透明程度，主要是从内含物的体积、位置和数量与瑕疵来评级。
    　　净度级别细分为FL/IF、VVS1、VVS2、VS1、VS2、SI1、SI2、P1、P2、P3共10个级别。
  3. 更多钻石知识 >
  1. 颜色 Color
  2. 　　颜色分级是指采用比色法，在规定的环境下对无色—浅黄色（褐、灰）系列的钻石进行颜色等级划分。
    　　最高色级为D，极其稀有；E-F级非常珍贵；G-H接近无色，性价比高；I-J可检测到轻微颜色，也属于价值较高的钻石。
  3. 更多钻石知识 >
  1. 切工 Cut
  2. 　　切工是指它的切磨比率的精确性和修饰完工后的完美性。
    　　好的切工应该尽可能的体现钻石的亮度和火彩，并且尽量保持原石重量。切工级别中，以“EX”为最优，以下分别是“VG”（非常好）及“G”（好）。

    手寸测量

    1.　准备一个宽约1厘米，长约10厘米的长方形纸条

    2.　将纸条绕在您需要佩戴的手指上，并移至手指最粗的部分（无论是指节或指根）

    3.　收紧纸条，但确保手指的舒适，不要过于压迫

    4.　在纸片上标记手指周长

    5.　对照下表即可得到您的手寸大小

    如何测量得更加准确：

    1.　想得到更确切的尺寸，您可以在19:00---21:00的时候测量，因为这个时候手指尺寸最为准确；

    2.　不要在天气过冷时测量手指，因为这个时候手指的尺寸是最小的；

    3.　根据季节不同来适当调整自己的号码，冬天购买戒指，由于天气较冷，手指比夏天要细一号到半号，戒指以带上后可以左右旋转但不易脱落为宜，夏天则以带上后感觉稍紧为宜；

    4.　您也可以前往附近的珠宝金店或格莱弗钻石商场，由专业人士进行更精确的测量。
